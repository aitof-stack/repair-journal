# Журнал заявок на ремонт оборудования

Веб-приложение для учета заявок на ремонт оборудования.

## Функции:
- Создание заявок на ремонт
- Управление пользователями с разными правами доступа
- Фильтрация и поиск заявок
- Экспорт данных
- Адаптивный дизайн для мобильных устройств
